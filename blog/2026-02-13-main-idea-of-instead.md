# Главный прикол Instead

Прикол Интстеда в том, что он принципиально по-другому работает.

Основная масса движков (Ren'py, Twine, QSP, URQ, Квестбук, Текстовые квесты, Meander (последний писк моды) — да тысячи их!) говорят тебе: у тебя есть локации -- это типа функции из ЯПов. Пиши в них инструкции в императивном (приказном) стиле:

```c
начало [
  выведиНаЭкран "Ты просыпаешься в 6 часов утра. Валяешься на кровати и думаешь, чем бы заняться"
  создайДействие "Встать" [
    перейтиВЛокацию хрустПозвоночника
  ]
  создайДействие "Полежать ёщо" [
    перейтиВЛокацию проспатьРаботу
  ]
]
хрустПозвоночника [
  выведиНаЭкран "В позвоночнике звонко хрустит, но кое-как тебе удается встать с кровати."
  создайДействие "Пойти на работу" [
    // todo
  ]
]
проспатьРаботу [
  выведиНаЭкран "Ты просыпаешь работу. Тебя торжественно увольняют с работы. Игре конец."
]
```

Инстед вообще пошел по принципиально другому пути: у тебя есть локации, в которых ты можешь размещать объекты и взаимодействовать с ними. Тебе нужно объявить их (напишу псевдокодом, чтобы донести смысол ||да, я забыл уже, как писать на Инстеде||):

```lua
локация твояКомната {
  название = "Твоя комната"
  начальныеОбъекты = [ты, кровать, работа]
}
объект кровать {
  описаниеВЛокации = fun() {
    if твоеСостояние == лежишьНаКровани {
      return nil
    }
    return "[Кровать|кровать] стоит посреди комнаты"
  }
  действиеПоНажатию = fun() {
    if твоеСостояние == лежишьНаКровани {

    }
  }
}
объект ты {
  описаниеВЛокации = fun() {
    if твоеСостояние == лежишьНаКровани {
      return "[Ты|ты] лежишь на [кровати|кровать]."
    }
    return "Ты стоишь посреди комнаты и думаешь о [работе|работа]"
  }
  действиеПоНажатию = fun() {
    if твоеСостояние == лежишьНаКровани {
      твоеСостояние = никакое
    }
  }
}
```
