# Омонимия (обзор)

Омоним — слово, имеющее разные значения.


* ключ — инструмент для открывания замков и одновременно «ключ» в музыке (звуковая высота).
* мир — вселенная и отсутствие войны.
Пила — инструмент для распиловки / прошедшее время от глагола «пилить» (она вчера пила водку)
* Банк — финансовое учреждение / берег реки.
* Лук — овощ / оружие для стрельбы.
* Молот — инструмент для забивания гвоздей / ударный музыкальный инструмент (молоток в оркестре).
* Ручка — часть двери / пишущий инструмент.
* Свет — видимое излучение / электрический ток (в разговоре «включить свет»).
* Ток — электрический ток / поток воды в реке
* Шторм — буря / резкое эмоциональное волнение.
* Яр — горячий источник / ярко‑красный цвет (в поэтическом употреблении).

    Коса:

    Прическа (заплетённые волосы)
    Сельскохозяйственный инструмент для скашивания травы
    Природная особенность рельефа (изогнутая полоса земли)
